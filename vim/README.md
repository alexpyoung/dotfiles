## It's been a while, let's get reacquainted with `vim`
- `:PlugUpdate`
- `less .vimrc`
- Files: `:ntf`, `m|?`
- Fuzzy Files: `:f`
- Search: `:rg PATTERN`
- Multicursor: `Ctrl-N`
- Shell: `:!COMMAND`
- Syntax: `:ss LANGUAGE`
