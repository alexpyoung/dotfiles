# dotfiles

![CI](https://github.com/alexpyoung/dotfiles/workflows/CI/badge.svg?branch=master)

## New Machine
1. `git clone git@github.com:alexpyoung/dotfiles.git`
2. `scripts/bootstrap/...`
   1. `brew`
   2. `vim`
   3. `zsh`
   4. `cron`
3. `scripts/setup/...`
   1. `macos`
   2. `git`
   3. `vscode`

## Existing Machine
- [ ] `scripts/update/brew`
- [ ] `scripts/update/cron`
