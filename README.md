# dotfiles

![CI](https://github.com/alexpyoung/dotfiles/workflows/CI/badge.svg?branch=master)

## New Machine
1. `git clone git@github.com:alexpyoung/dotfiles.git`
2. `bin/bootstrap/...`
   1. `brew.sh`
   2. `vim.sh`
   3. `zsh.sh`
   4. `cron.sh`
3. `bin/setup/...`
   1. `macos.sh`
   2. `git.sh`
   3. `vscode.sh`

## Existing Machine
- [ ] `bin/update/brew.py`
- [ ] `bin/update/cron.sh`
